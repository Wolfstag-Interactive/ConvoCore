Detailed Description

World-space dialogue UI for 3D character conversations.

No sprite support; no slot system. Character placement is entirely presence-driven via ConvoCoreCharacterPresence.

Characters are persistent across lines. Expression application is the only per-line operation; characters are not despawned and re-spawned between lines.

ConvoCoreCharacterPresence.OnConversationBegin is called when ConvoCore.StartedConversation fires. ConvoCoreCharacterPresence.OnConversationEnd is called when ConvoCore.EndedConversation fires. The spawner is never called directly by this UI for release; that responsibility belongs to the presence.
