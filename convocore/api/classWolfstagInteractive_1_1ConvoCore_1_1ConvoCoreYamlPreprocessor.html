Detailed Description

Three-phase YAML preprocessor that replaces the old regex-based EnsureQuotesOnLocalizedValues:

  1. Pre-flight — scans the raw text for file-level problems (BOM, git conflict markers, multiple documents, tab indentation, smart quotes, zero-width characters) before any transforms are applied.

State machine transform — walks the file line-by-line, enters LocalizedDialogue blocks by key name, and applies a safe escape pipeline only to locale value lines. All other lines pass through unchanged. Column offset recording — for every transformed line, records how many characters were added so that YamlDotNet's error columns can be corrected back to the original file coordinates.
