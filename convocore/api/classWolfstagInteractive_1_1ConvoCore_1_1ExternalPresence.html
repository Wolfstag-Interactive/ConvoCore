Detailed Description

Presence type for characters that are fully managed by the developer.

ConvoCore never spawns, parents, or destroys anything.

  • Scene-resident characters: resolved via ConvoCoreSceneCharacterRegistry on the spawner.
  • Spawn-from-prefab characters: returns null (the developer is responsible for placing and managing these).

OnConversationEnd is a no-op. The developer is fully responsible for character lifecycle in both source modes.

Use case: characters already placed in the world by the developer, with no ConvoCore lifecycle involvement.
