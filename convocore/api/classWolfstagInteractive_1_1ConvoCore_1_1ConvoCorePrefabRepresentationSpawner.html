Detailed Description

Resolves, tracks, and releases prefab-based character displays for a single UI instance.

Supports two resolution paths determined by CharacterSourceMode on the representation asset:

  • CharacterSourceMode.SpawnFromPrefab – instance is drawn from ConvoCorePrefabPool and returned on release.
  • CharacterSourceMode.SceneResident – instance is located via ConvoCoreSceneCharacterRegistry. ConvoCore never spawns, pools, or destroys it.

Active entries are tracked by the Transform passed as the slot anchor. Passing the same Transform again releases the previous occupant before placing a new one. Scene-resident characters resolved via TryGetSceneResident are not tracked by slot and are never pooled or destroyed by this spawner.
