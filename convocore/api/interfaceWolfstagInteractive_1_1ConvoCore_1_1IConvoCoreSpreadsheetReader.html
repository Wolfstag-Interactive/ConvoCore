Detailed Description

Defines the contract for reading a spreadsheet file into ConvoCore's internal dialogue configuration format.

Sheet name = ConversationKey. Header row columns must include the CharacterID and LineID column names as configured in ConvoCoreSettings. At least one language code column is required per sheet. Unrecognized columns are preserved during writeback but ignored during import. Sheets whose names begin with the configured skip prefix are ignored.
