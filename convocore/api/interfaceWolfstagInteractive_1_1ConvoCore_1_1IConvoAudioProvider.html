Detailed Description

Interface for audio playback backends.

Implement to support FMOD, Wwise, or any custom audio middleware. The built-in implementation is ConvoCoreUnityAudioProvider. Assign via the ConvoCore inspector or call ConvoCore.SetAudioProvider at runtime.

Play a voice line for the given dialogue line.

The full ConvoCoreConversationData.DialogueLineInfo is passed so middleware providers can use LineID, characterID, or expression data to route to the correct event or bank. Providers that use Unity AudioClips should cast reference to ConvoCoreUnityAudioReference. Providers using middleware should ignore reference and use line.LineID as the event key.

Implemented in WolfstagInteractive.ConvoCore.ConvoCoreUnityAudioProvider.
