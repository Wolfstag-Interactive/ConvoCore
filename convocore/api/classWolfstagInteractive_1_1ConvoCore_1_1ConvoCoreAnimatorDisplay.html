Detailed Description

Prefab display component that maps expression display names to Animator parameters.

Expression display names are matched against PrefabExpressionMapping.DisplayName on the bound PrefabCharacterRepresentationData asset at bind time, building a runtime GUID-to-parameter lookup. Warns by display name when a mapping is missing.

Supports Bool, Int, Float, and Trigger parameter types.
