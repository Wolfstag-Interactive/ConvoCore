Detailed Description

Canvas-space dialogue UI supporting both sprite and prefab-in-canvas character representations.

Mixed representation support: a single line may have sprite characters in some slots and prefab characters in others. Slot assignment uses DialogueLineDisplayOptions.SlotId (named addressing) first, then index-based fallback using characterSlotAnchors.

Owns its own ConvoCorePrefabRepresentationSpawner and slot anchor list. Slot count is determined by the length of characterSlotAnchors; there is no hardcoded cap.
