◆ AudioBackend

enum WolfstagInteractive.ConvoCore.AudioBackend

Selects the audio playback backend for a ConvoCoreAudioManifest.

Controls which fields are shown in the manifest inspector and how the runner resolves and plays audio at runtime.

Enumerator
UnityAudioSource 

Plays audio via Unity's built-in AudioSource.

Assign UnityEngine.AudioClip directly in each manifest entry slot. No extra components are required — the runner auto-provisions an AudioSource on its GameObject when none is manually assigned.

FMOD 

Audio is driven by FMOD Studio.

The runner triggers events using ConvoCoreConversationData.DialogueLineInfo.LineID as the event key. Assign an FMOD adapter that implements IConvoAudioProvider to the ConvoCore runner. AudioClip slots are not shown in the manifest inspector.

Wwise 

Audio is driven by Wwise.

The runner posts events using ConvoCoreConversationData.DialogueLineInfo.LineID as the event key. Assign a Wwise adapter that implements IConvoAudioProvider to the ConvoCore runner. AudioClip slots are not shown in the manifest inspector.

Custom 

A custom IConvoAudioProvider drives playback.

Assign the provider component to the ConvoCore runner. Both AudioClip and ConvoAudioReference slots are shown in the manifest inspector.

◆ ConversationPresentationMode

enum WolfstagInteractive.ConvoCore.ConversationPresentationMode

Controls whether a dialogue line displays text, plays audio, or both.

Set per-line via ConvoCoreConversationData.DialogueLineInfo.PresentationMode or as a conversation-level default via ConvoCoreConversationData.DefaultPresentationMode.

Enumerator
AudioAndText 

Text is displayed in the UI.

Audio plays if a manifest and provider are assigned.

AudioOnly 

No text is sent to the UI.

Progression is driven by audio length or explicit input.

TextOnly 

Text is displayed.

Audio is not played even if a manifest and provider are present.

◆ ExcelFormulaCellBehavior

enum WolfstagInteractive.ConvoCore.ExcelFormulaCellBehavior

Controls how ConvoCore handles formula cells encountered during Excel spreadsheet import.

Enumerator
UseCachedValue 

Use the cached result value stored in the cell.

This is the default and works for most cases where the file was saved with calculations up to date.

TreatAsError 

Treat any formula cell as a parse error and abort import of that sheet.

SkipRow 

Skip rows that contain any formula cell silently without aborting the import.
