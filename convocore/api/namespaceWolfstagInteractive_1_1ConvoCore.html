◆ CharacterSourceMode

enum WolfstagInteractive.ConvoCore.CharacterSourceMode

Determines how ConvoCore resolves the live IConvoCoreCharacterDisplay instance for a PrefabCharacterRepresentationData asset.

Enumerator
SpawnFromPrefab 

ConvoCore instantiates the prefab via the pool and owns the full lifecycle.

The character is spawned when needed and returned to the pool when released.

SceneResident 

ConvoCore locates an already-existing scene instance via a ConvoCoreSceneCharacterRegistry.

ConvoCore never spawns, pools, or destroys the instance. The developer is fully responsible for the character's lifecycle.

◆ ExcelFormulaCellBehavior

enum WolfstagInteractive.ConvoCore.ExcelFormulaCellBehavior

Controls how ConvoCore handles formula cells encountered during Excel spreadsheet import.

Enumerator
UseCachedValue 

Use the cached result value stored in the cell.

This is the default and works for most cases where the file was saved with calculations up to date.

TreatAsError 

Treat any formula cell as a parse error and abort import of that sheet.

SkipRow 

Skip rows that contain any formula cell silently without aborting the import.
