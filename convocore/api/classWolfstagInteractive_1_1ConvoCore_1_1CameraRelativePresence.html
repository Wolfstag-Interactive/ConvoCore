Detailed Description

Presence type that places characters at authored offsets relative to the active camera.

Two positioning modes:

  • Once — position is calculated once on ResolvePresence and not updated.
  • Continuous — a ConvoCoreCameraRelativePosition component is attached to the spawned character and updates its world position every frame.

Use case: first-person games, VR, or any setup where character position should be relative to the player's view.
