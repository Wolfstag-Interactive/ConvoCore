Detailed Description

Prefab display component that maps expression display names to SkinnedMeshRenderer blend shape indices and target weights.

Supports optional smooth blending over time.

Expression display names are matched against PrefabExpressionMapping.DisplayName on the bound PrefabCharacterRepresentationData asset at bind time, building a runtime GUID-to-blend-shape lookup.
