Detailed Description

Add this component to any scene GameObject that should be driven by ConvoCore as a scene-resident character (i.e.

not spawned from a pool).

The component resolves an IConvoCoreCharacterDisplay from this GameObject or its children and registers it with the assigned ConvoCoreSceneCharacterRegistry on enable, unregistering on disable.

The CharacterId must match the Scene Character ID set on the PrefabCharacterRepresentationData asset configured in SceneResident mode.
