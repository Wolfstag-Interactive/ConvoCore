Detailed Description

Minimal prefab display component.

All expression logic is delegated to BaseExpressionAction ScriptableObjects defined on the PrefabCharacterRepresentationData asset. No built-in visual change behaviour is applied by this component itself.

Use this when you want full control over expression results via ScriptableObject actions and do not need built-in Animator, blend-shape, or sprite handling.
