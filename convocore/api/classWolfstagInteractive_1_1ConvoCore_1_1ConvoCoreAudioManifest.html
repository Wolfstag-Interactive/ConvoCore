Detailed Description

Maps dialogue line IDs to audio clips or middleware references, supporting Unity AudioSource, FMOD, Wwise, and custom provider backends.

Assign to ConvoCoreConversationData.AudioManifest to enable voice playback.

Resolve a ConvoAudioReference for a given line ID and language.

Only returns a value when an entry has a non-null AudioEntry.Reference assigned. Tries exact locale match first, then language-agnostic fallback. For the common Unity AudioSource path, use ResolveClip instead.
